Problem It Solves 

In 2–3 sentences: what challenge does your app address, and who is the target user? 

It gives a space for the users to get organized. As well as targets users habits to suggest organization throughout their day. 

Page 1 — Name & Description 

Home page; Shows the whole week with habits and a completion circle  

 

Page 2 — Name & Description 

Day overview; Shoes your events on a specific day, and has a way to add events  

 

Page 3 — Name & Description 

Adding events; This page will be the way to add an event on a specific day, with all the tools to make it as personalized as possible  

 

Page 4 — Name & Description (optional but recommended) 

Completed Calender; Shows all the events planned with color coordination  

 

Key Interactive Feature 1 

Uses habits that you implement to give you suggestions and reminders on how to approach your daily tasks. 

Key Interactive Feature 2 

Uses a completion circle to mark how many task you have completed and shows you a percentage.  

 

Key Interactive Feature 3 

Add event button 

 

Key Interactive Feature 4 

Can select a habit when adding an event to make the app know what type of event is
